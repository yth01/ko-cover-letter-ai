# 한국어 자기소개서 AI
- 핵심 역량 중심의 채용 트렌드와 개인의 직무 성향을 파악하는 것이 중요해지고 있다. 따라서 이 서비스를 통해 다양한 측면으로 자기소개서를 분석하고 숨겨진 인사이트를 발견할 수 있도록 돕는다.
